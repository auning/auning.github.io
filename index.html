🌐 第五步：部署到GitHub Pages

创建GitHub仓库

  1. 登录GitHub，点击右上角的 + → New repository
  2. 仓库名必须为：yourusername.github.io（替换为您的GitHub用户名）
  3. 设置为Public
  4. 不要初始化README、.gitignore或license
  5. 点击 Create repository

配置SSH密钥（推荐方式）

生成SSH密钥

添加公钥到GitHub

  1. 登录GitHub → Settings → SSH and GPG keys
  2. 点击 New SSH key
  3. Title填写：Hexo Blog
  4. Key粘贴刚才复制的公钥内容
  5. 点击 Add SSH key

测试SSH连接

看到类似 Hi username! You've successfully authenticated... 的消息即为成功。

📝 日常使用速查表

操作 命令 说明
新建文章 hexo new "标题" 在_posts目录创建文章
新建草稿 hexo new draft "标题" 创建草稿，不会发布
发布草稿 hexo publish "标题" 将草稿移动到_posts
本地预览 hexo s 启动本地服务器
生成静态文件 hexo g 生成public目录
清除缓存 hexo clean 清除缓存和生成文件
部署 hexo d 部署到远程仓库
一键发布 hexo clean && hexo g && hexo d 清除→生成→部署
